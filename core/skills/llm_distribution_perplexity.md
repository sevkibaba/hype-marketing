# LLM Distribution - Perplexity

Skill for getting featured in Perplexity answers.

## Key Facts
- Three-layer ML system (L3 XGBoost quality gate)
- Prefers no sources over low-quality ones
- Rewards **external authority**, not website SEO
- 30-day freshness window (40% weight)
- Reddit dominates citations (6.6%)
- Only 11% overlap with ChatGPT

## Prerequisites
- [ ] Earned media presence (PR, press)
- [ ] Reddit/forum activity
- [ ] Fresh content (30-day updates)
- [ ] FAQ schema

## Strategy

### 1. Build External Authority
- Get featured in Tier-1 publications
- Active Reddit participation (not promotional!)
- GitHub, LinkedIn, Amazon presence
- Earned media > owned media for Perplexity

### 2. Content Optimization
- **BLUF format** - Answer first, then context
- 40-60 word extractable answer blocks
- Clear entity identification (name, category, differentiation)
- FAQ schema = 3.2x more likely to appear

### 3. Technical Requirements
- Allow PerplexityBot crawlers
- Fresh content (update regularly)
- No AI-generated content penalty
- Expert bylines and credentials

### 4. What Works
- News/earned media mentions
- Reddit discussions (helpful, not promotional)
- Original research and data
- Clear, factual content

## Quick Wins
- Add FAQ schema to content
- Start Reddit helpful participation
- Publish original research/data
- Update content within 30 days
- Get mentioned in 2+ news publications
