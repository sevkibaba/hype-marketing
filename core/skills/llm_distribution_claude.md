# LLM Distribution - Claude

Skill for getting featured in Claude recommendations.

## Key Facts
- No real-time web browsing - builds from training data
- Prioritizes brands with frequent mentions in authoritative content
- MCP integrations can become default recommendations
- No paid placements

## Prerequisites
- [ ] Comprehensive content (docs, guides, case studies)
- [ ] llms.txt for documentation
- [ ] Schema markup
- [ ] MCP server integration (optional but powerful)

## Strategy

### 1. Build Authority in Training Data
- Get mentioned in educational content
- Create comparison content
- Third-party reviews and analysis
- Technical documentation

### 2. Content Types That Work
- Comparison articles ("X vs Y")
- How-to guides and tutorials
- Case studies with real metrics
- Technical documentation
- "Best X for Y" lists from authoritative sources

### 3. MCP Integration
If you build an MCP server:
- Claude can use your tool in real-time
- Becomes default when users ask for tasks in your category
- Real-time data access without training data dependency

### 4. Technical SEO
- Implement Schema markup
- Create llms.txt for documentation
- Ensure cross-source consistency (same info across multiple trusted sources)
- Add expert bylines and credentials (E-E-A-T signals)

## Quick Wins
- Publish comparison content
- Get mentioned in 3+ educational publications
- Create comprehensive documentation
- Implement llms.txt
