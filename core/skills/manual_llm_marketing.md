# Manual LLM Marketing

Skill for human-driven marketing to get into LLM results.

## Core Insight
AI models cite brands that demonstrate authority through **human-created content**. 
Automation doesn't build the trust signals AI models value.

## Key Techniques

### 1. Content Creation
- **Blog content** targeting question-based queries
- **Newsletters** building loyal audiences
- **Case studies** with concrete proof
- Focus on questions people actually ask

### 2. Community Building
- **Reddit**: 90/10 rule - 90% helpful, 10% brand
- **Discord**: Build genuine community
- **Niche forums**: Active participation
- Be helpful, not promotional

### 3. PR & Media
- **HARO/Qwoted**: Respond to journalist queries
- **Industry publications**: Guest posts
- **Original research**: Most cited by AI
- **Wikipedia**: Eligible for notable brands

### 4. Events
- Speaking at conferences
- Sponsorship opportunities
- Host own events
- Build in-person relationships

### 5. Partnerships
- Affiliate programs
- Strategic co-marketing
- Review programs
- Referral networks

## Quick Wins (Today)
- [ ] Check robots.txt (2 min) - allow AI crawlers
- [ ] Restructure content to answer questions in first paragraph
- [ ] Start Reddit participation (helpful, not promotional)
- [ ] Sign up for HARO
- [ ] Publish original research/data

## What Doesn't Work
- Pure automation without substance
- Buying links or mentions
- Content without genuine value
- Ignoring community building
