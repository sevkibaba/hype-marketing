# Automated LLM Marketing

Skill for AI-driven marketing to get into LLM results.

## Core Insight
Automation can scale certain tactics but cannot replace human authority building.
Use AI to amplify, not replace, genuine value creation.

## Key Techniques

### 1. AI Content Generation
- Use tools like Jasper, GPT-4 for scale
- Human oversight required for quality
- Generate first drafts, human-edit for quality
- Maintain brand voice consistency

### 2. Automated Social Media
- Tools: SocialPost.ai, FlowPost, Publishr
- Cross-platform scheduling
- Content repurposing automation
- Monitor engagement

### 3. AI-Powered SEO
- SchemaGenAI for schema markup
- Alli AI for optimization
- Pagesmith.ai for page optimization
- Automated JSON-LD generation

### 4. Automated Outreach
- Outreach.io, Reply.io for email automation
- Zapier integrations
- Automated follow-up sequences
- Personalization at scale

### 5. Bot-Based Engagement
- Fleece AI for community automation
- Discord/Slack AI automation
- Automated response systems
- Monitor but don't spam

## GEO (Getting Cited in LLM Results)

### What AI Engines Look For
- Clear, structured content
- Answer-first format
- Fact-based assertions with sources
- Consistent information across sources

### Practical Steps
1. Implement structured data (Schema)
2. Create FAQ content
3. Build comparison content
4. Add citations and references
5. Maintain consistency across all content

### Tracking Tools
- Brand mentions in AI responses
- Referral traffic from AI
- SERP tracking for AI mentions
- Sentiment analysis
