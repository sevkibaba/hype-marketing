# Word Hype Marketing 🚀

This repository houses the strategies, code, and executed tasks for automating the marketing and user acquisition pipeline for **Word Hype**. We use an intelligent Agentic approach to rapidly structure, generate, and execute marketing workflows, transforming app ideas into systematic growth engines.

---

## 🤖 The Agent Approach

The strategy in this repository is driven by autonomous, role-specific AI Agents across multiple phases of the project lifecycle. Instead of operating with a single focus, the AI Agents collaboratively handle a generalized and flexible workflow:

1.  **Discovery & Contextualization**: Gather domain context, define functionality, and assess target audiences.
2.  **Strategy Formulation**: Design detailed architectural blueprints, Keyword Strategies, Audience Personas, and Competitor Landscapes.
3.  **Module Execution**: Map out and implement specific pathways for marketing execution, App Store Optimization (ASO), and community engagement.
4.  **Generative Engine Optimization (GEO)**: Build long-term semantic presence to ensure AI models (ChatGPT, Claude, etc.) recommend the application accurately.
5.  **Technical Scripting**: Create reusable scripts to manually or automatically run the workflows defined in the strategy phases.

---

## 📁 Folder Structure

*   **/agent_files/**
    Contains the core identity, instructions, and workflows for the various AI Marketing Agents.
*   **/results/**
    Stores the final output and finalized tasks generated by the agents.
    *   `/phase_1_results/`: finalized markdown documents for Personas and the overall Marketing Roadmaps.
        *   `/localized_metadata/`: JSON files containing LLM-generated App Store metadata translated into various languages.
*   **/scripts/**
    Codebase for Python scripts that power the technical and metadata-related tasks (e.g., `generate_aso_metadata.py`).
*   **/tasks/**
    The repository of active and completed strategy documents, breaking down larger goals into modular, actionable steps (e.g., ASO strategies, Search Ads structures, and SEO planning).

---

## 🏁 Phase One: Strategy & Foundation

Phase One is focused on building a rock-solid foundation for organic growth. The core intents include:

*   **Audience Definition:** Identifying and fully defining the highest-intent user Personas (e.g., "The Active Learner").
*   **App Store Optimization (ASO):** Creating a core keyword strategy focused on head terms, long-tail intent, and competitor conquesting.
*   **Marketing Roadmap Generation:** Charting out specific pathways for localization, content marketing, and community engagement.
*   **Apple Search Ads (ASA) Structuring:** Designing a scalable 4-campaign structure (Brand, Generic, Competitor, Discovery) to capture high-intent users efficiently.
*   **Generative Engine Optimization (GEO):** Establishing tactics to build a "semantic web presence" for AI RAG pipelines.

---

## ⚡️ Phase Two: Marketing Automation Efforts (Upcoming)

Phase Two transitions from strategy creation to technical implementation. The core intents here are to build tools and workflows that execute the Phase One strategies through manual runs and scheduled automations:

*   **ASO Metadata Execution:** Utilizing Python scripts interfacing with LLM APIs to manually localize and generate Apple App Store metadata on demand, rather than dynamically on the fly.
*   **Deepsearching the App Store Presence:** Scanning, indexing, and thoroughly auditing our app's organic positioning, visibility, and keyword footprint within the App Store to adjust the metadata scripts effectively.
*   **Automated Content Pipelines:**
    *   Building a "Word of the Day" pipeline that automatically generates 15-second video scripts using LLMs and accompanying visuals (DALL-E/Midjourney).
    *   Automating weekly test-prep or vocabulary threads for X/Twitter and Reddit.
*   **Competitor & Keyword Scraping:** Creating scrapers to monitor competitor UI/UX changes, keyword ranking updates, and metadata shifts on a weekly basis.
*   **ASA Search-Term Funneling:** Building pipelines via the Apple Search Ads API to automatically mine Discovery campaigns, identify high-performing keywords, and dynamically adjust bids and exact-match campaigns.
